Fertilizer Recommendation System

A machine learning–based web application that recommends the most suitable fertilizer based on soil properties, crop type, and environmental conditions.

This project uses a Random Forest Classifier trained on agricultural data and is deployed as a free interactive Streamlit demo app.

Features

Built with machine learning using Random Forest
Interactive Streamlit web interface
